Tipos de position

Position define como se posiciona un elemento a otros o al viewport.

Position static

Es la posicion por defecto
No funciona el top, left...

Position relative

Se posiciona respecto a si mismo
Mantiene su espacio original
Muy usada como referencia para hijos.

Position absolute

Sal del flujo normal
Se posiciona respecto a el padre con position distinta a static o al body si no hay padre.

Padre
Hijo

Position fixed

Se posiciona respecto al viewport
No se mueve al hacer scroll
Ideal para headers y botones flotantes.

1
2
3

Position sticky

Mezcla de relative + fixed
Se pega al llegar a cierto punto del scroll
Necesita top, left...
No funciona si el contenedor tiene overflow hidden.

Resumen

Static: normal
Relative: mueve sin romper
Absolute: flota
Fixed: fijo en la pantalla
Sticky: pegajoso con scroll
