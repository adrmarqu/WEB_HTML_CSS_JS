Tipos de display

Display define como se comporta un elemento en el layout: si ocupa línea completa, si se alinea con otros, si puede tener ancho/alto, etc.

Display block

Ocupa todo el ancho disponible
Empieza en una nueva linea
Acepta width, height, margin, padding.

Display inline

Se comporta como texto
No salta de linea
No acepta width ni height
Solo acepta margin y padding horizontal.

Display flex

Activa Flexbox
Controla la distribución en 1 dimension (fila o columna)
Ideal para layouts y alineaciones.

Display grid

Activa CSS Grid
Controla layouts en 2 dimensiones (filas y columnas)
Perfecto para estructuras grandes.

Display inline-flex

Igual que flex, pero el contenedor se comporta como inline.

Display inline-grid

Igual que grid, pero en linea.

Resumen
