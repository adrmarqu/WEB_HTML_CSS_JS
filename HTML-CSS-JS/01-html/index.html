Etiquetas semánticas estructurales

div: Etiqueta estructural pero no semántica
header: Cabecera de un sitio web o una sección
main: Define la sección principal del documento, sólo puede existir una etiqueta main por documento
footer: Pié de página de un sitio web o de una sección
nav: Representa una navegación
section: Representa una sección de contenido genérico
article: Representa contenido dependiente de una sección
aside: Representa contenido complementario o secundario
address: Muestra información de contacto

Etiquetas de bloque vs Etiquetas de línea

La etiqueta de bloque por excelencia es la div, una etiqueta de bloque es aquella que genera saltos de línea con sus etiquetas hermanas.

La etiqueta de línea por excelencia es la span, una etiqueta de línea sólo ocupa el espacio necesario que tiene si contenido. Hola soy otra span.

Scripts en Html

Para incluir javascript y dar funcionalidades al documento html. Usaremos la etiqueta <script>. El script puede ir en el head del documento, o dentro del body. Si la incluimos en el body es preferible que esté al final mismo.

La etiqueta <script> puede contener javascript, o vincular un archivo externo de javascript.

Imágenes con Html

En html podemos utilizar diferentes tipos de imágen dependiendo de su compresión; sin perdida de calidad: GIF, APNGm WEBP y PNG, y con su perdida de calidad cómo JPG.

GIF: Es un formato sin perdida con capacidad para comprimir hasta 256 colores, válido para tintas planas, logotipos, etc... Acepta animaciones y acepta transparencia en alguno de sus colores.

PNG-8: Igual a GIF y su ratio de compresión es mejor que el archivo GIF. No acepta aniimaciones.

APNG: Igual a GIF pero que acepta diferentes ratios de compresión de 16 y 24 bits. Acepta animaciones.

WEBP: Puede comprimir imágnes con pérdida y sin pérdida y con mejores ratios que GIF, PNG o JPG.

JPG: Formato perfecto para cuatricomías e imágen forográfica, pero tiene pérdida de calidad.

PNG-24: Formato perfecto para fotografías y transparencias a color real, tiene muy poca pérdida de calidad.

SVG: Formato de imágen vectorial, sin pérdida, y la más recomendable para logotipos y pictogramas.
