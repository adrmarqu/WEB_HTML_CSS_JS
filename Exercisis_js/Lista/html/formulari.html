Eventos y formularios con Javascript y Cookies

Ejercicio 1

Crea un formulario con HTML y css con los siguientes campos: usuario, telefono, nacimiento(dia, mes, año), email, dni, checkbox de acepto condiciones, registrate, limpia formulario, guardar datos, recuperar datos, textarea de información del formulario. Este formulario tendrá estas características:
1- No se ha de enviar si el usuario no acepta las condiciones.
2- Cuando el usuario escriba una letra en el input, se muestre un mensaje en el textArea o debajo del input, que tenga el valor del input y si su valor es válido.
3- Si alguno de los inputs es incorrecto, el formulario no se ha de enviar.
4- El inputNombre no puede comenzar por un número y ha de tener una longitud de 3 a 20 carácteres.
5- El inputTelef ha de tener un formato de teléfono móvil válido.
6- El inputDiaMesAño ha de tener una fecha válida y ser una persona mayor de edad.
7- El inputEmail ha de tener un formato de correo electróncio con una @ y un punto.
8- el inputDNI ha de tener un formato DNI con 8 números y una letra mayúscula o minúscula, o un NIE con letra inicial y final. La validación del pattern ha de ser auténtica.
9- El textArea ha de tener al menos 10 carácteres.
10- Cuando le des a "Guardar Gatos", se guardaran todos los inputs en una cookie (cada dato en una cookie)
11- Cuando le des a "Recuperar Datos", los valores de los inputs se sustituiran por los datos guardados en las cookies.
12- Todos los inputs del formulario han de estar llenos.
13- Si no hay una cookie para un valor, se muestre en el input correspondiente con un valor en blanco.
