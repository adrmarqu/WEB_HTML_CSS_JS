DOM y LocalStorage

Ejercicio 1

Crea una agenda de contactos utilizando HTML, Bootstrap y Javascript, donde los datos se guarden de manera persistente mediante el localStorage del navegador.

Cuando se recargue la página, los contactos han de continuar existiendo.

Requisitos

Formulario

Ha de haber un formulario con los siguientes puntos:
- Un campo de texto para un nombre.
- Un campo de texto para un telefono móvil.
- El teléfono ha de tener 9 digitos.

Guardar contacto

Cuando se pulse el botón 'Guardar':
- Se han de validar los datos: Que no este vacio, que el teléfono tenga 9 digitos y que los nombres no se puedan repetir.
- El nombre sera la clave del localStorage.
- El telefono será el valor del localStorage.
- Se ha de mostrar un mensaje de exito o error.

Recuperar contacto

Cuando se pulse el botón 'Recuperar':
- Se buscará el contacto por nombre.
- Si existe: Se cargará el teléfono en el formulario y el contacto pasara a modo edición.
- Si no existe: Se mostrará un mensaje de error.

Editar contacto

Cuando estes en modo edición:
- Se podrá modificar el nombre y/o el teléfono.
- Al guardar: Si el nombre cambia, se ha de eliminar el contacto antiguo y guardar el nuevo.

Eliminar contacto

Cuando se pulse el botón 'Eliminar':
- Se eliminará el contacto indicado por el nombre.
- Si se estaba editando, el formulario se ha de limpiar.
- Se ha de actualizar la lista.

Borrar toda la agenda

Cuando se pulse el botón 'Borrar todo':
- Se han de elminar todos los contactos.
- Se ha de limpiar el formulario.
- Se ha de mostrar un mensaje de confirmación.

Lista de contactos

- Mostrará todos los contactos en una lista.
- Han de estar ordenados alfabeticamente por el nombre.
- Si no hay contactos hay que mostrar el siguiente mensaje: "Sin contactos en la agenda".
