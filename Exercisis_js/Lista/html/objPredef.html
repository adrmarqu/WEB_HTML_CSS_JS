Objetos predefinidos

Ejercicio 1

Utiliza Javascript para crear a HTML una tabla con estos datos: [Valor màxim que pot tenir un número en JavaScript, Valor mínim que pot tenir un número en JavaScript, Alçada total de la pantalla, Amplada total de la pantalla, Alçada de la pàgina web, Amplada de la pàgina web, URL de la pàgina web, Nom de la pàgina web amb la seva extensió (index.html), Nom de la pàgina web sense la seva extensió (index), Un valor aleatori entre 0 i 200 (utilitza Math), Sistema operatiu de l’ordinador].

Ejercicio 2

Utiliza el javascript para detectar si el navegador tiene las cookies activadas. En caso de estar activadas informa al usuario con un confirm, si el usuario le da a cancel, redirigirlo a otra página.

Ejercicio 3

Pide mediante un prompt un título para la página web. Si el usuario no introduce nada o pulsa a cancelar, vuelve a pedirlo hasta que introduzca un valor valido. Para acabar, utiliza un metodo de document para establecer el valor en mayuscula como el nuevo título de la página.

Ejercicio 4

Pide mediante un prompt 2 valores separados por una coma. Si són incorrectos, indica al usuario como ha de escribirlo correctamente: títol ha de contener entre 3 y 20 carácteres, y barra ha de contener "yes" o "no",

Ejercicio 5

Utiliza los valores anteriores para abrir mediante el objeto window de JavaScript una ventana nueva a finestra.html. Si barra es "yes" meter barras de desplazamiento y unas medidas aleatorias entre 50 y 400 de altura y anchura.

Ejercicio 6

Modifica el título de finestra.html para que su valor siga siendo el título guardado en la variable de titol del ejercicio 4.

Ejercicio 7

Muestra en finestra.html la hora, los minuto y los segundos, más un mensaje de buenos días si la hora es de 6 a 14, un mensaje de buenas tardes si la hora es mayor que 14 y menor que 20, y un mensaje de buenas noches en el caso contrario.

Ejercicio 8

Pedir por prompt una nueva dirección HTML donde pida una nueva dirección HTML. Tendrás de cambiar la dirección de la ventana principal de objetos predefinidos por la URL introducida.
